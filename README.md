Este é um jogo básico e simples de advinhação em Python. Podendo apresentar erros simples de sintaxe mas que não comprometem a lógica geral
do programa por trás de seu funcionamento.

Foi desenvolvido durante cerca de 15 dias em Python, a fim de que eu pudesse estudar a linguagem em seus principais fundamentos e pudesse
evoluir com o passar do tempo.

É o primeiro projeto "grande" (tendo em vista os anteriores) na linguagem em questão, feito unicamente para fins acadêmicos.
